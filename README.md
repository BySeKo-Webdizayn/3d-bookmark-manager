## ✨ Özellikler

- 🌌 **3D Görselleştirme**: Yer imlerinizi 3D uzayda görün
- 🔍 **Otomatik Çift Tespiti**: Tek tıkla çift yer imlerini bulun
- 🗑️ **Toplu Silme**: Çiftleri toplu olarak silin
- 📥 **İçe/Dışa Aktarma**: JSON ve HTML formatlarında yedekleme
- 🌐 **Çapraz Tarayıcı**: Chrome, Firefox, Edge desteği

## 🚀 Kurulum

### Chrome Web Store
1. [Chrome Web Store](https://chrome.google.com/webstore/detail/YOUR_EXTENSION_ID) sayfasına git
2. "Add to Chrome" butonuna tıkla
3. Hazır!

### Firefox Add-ons
1. [Firefox Add-ons](https://addons.mozilla.org/firefox/addon/YOUR_EXTENSION_ID) sayfasına git
2. "Add to Firefox" butonuna tıkla

### Edge Add-ons
1. [Edge Add-ons](https://microsoftedge.microsoft.com/addons/detail/YOUR_EXTENSION_ID) sayfasına git
2. "Get" butonuna tıkla
